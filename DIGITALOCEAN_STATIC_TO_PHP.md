# Как изменить Static Site на PHP Web Service в DigitalOcean

## Проблема
Ваш компонент настроен как **"Static Site"**, а для работы PHP (админ-панель) нужен **"Web Service"**.

## Решение: Изменить тип компонента

### Вариант 1: Через веб-интерфейс (рекомендуется)

1. **В настройках компонента** (где вы сейчас):
   - Нажмите кнопку **"Edit"** рядом с любым полем (например, "Source")
   - Или найдите кнопку **"Change Component Type"** / **"Edit Component"**

2. **Измените тип компонента:**
   - Найдите опцию **"Component Type"** или **"Type"**
   - Измените с **"Static Site"** на **"Web Service"**
   - Выберите **"PHP"** как runtime

3. **После изменения типа появятся новые поля:**
   - **Run Command** - здесь укажите: `php -S 0.0.0.0:8080 -t . router.php`
   - **HTTP Port** - укажите `8080`

### Вариант 2: Удалить и создать заново

Если нельзя изменить тип:

1. **Удалите текущий компонент** (Static Site)
2. **Создайте новый компонент:**
   - Type: **Web Service**
   - Runtime: **PHP**
   - Source: тот же репозиторий `SMaliskins/TRAVEL-GTR`
   - Branch: `main`
   - **Run Command**: `php -S 0.0.0.0:8080 -t . router.php`
   - **HTTP Port**: `8080`

### Вариант 3: Добавить второй компонент (Web Service)

Оставьте Static Site для главной страницы, добавьте Web Service для админки:

1. В настройках App нажмите **"Add Component"**
2. Выберите **"Web Service"**
3. Настройте:
   - Source: тот же репозиторий
   - Branch: `main`
   - **Run Command**: `php -S 0.0.0.0:8080 -t . router.php`
   - **HTTP Port**: `8080`
   - **Routes**: добавьте маршрут `/admin/*` → этот компонент

## После изменения

После того как компонент станет Web Service:
1. Появится поле **"Run Command"**
2. Введите: `php -S 0.0.0.0:8080 -t . router.php`
3. Укажите **HTTP Port**: `8080`
4. Сохраните и перезапустите приложение

## Альтернатива: Использовать только статические файлы

Если не хотите менять тип, можно:
- Оставить главную страницу как Static Site
- Создать отдельный Web Service только для `/admin/`

Но проще всего - изменить тип на Web Service для всего приложения.
